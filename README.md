## How to Use

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd school-website
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - No server required for local testing

3. **Deploy to web**
   - Upload all files to your web hosting
   - Maintain the folder structure (especially `scr/` folder)

## Customization

### Change School Name
Edit in `index.html`:
- Navigation logo
- Hero section title
- Meta title tag

### Add More Images
1. Add images to the `scr/` folder
2. Add new carousel items in `index.html`
3. JavaScript automatically handles dot generation

### Change Colors
Edit `styles.css`:
- Main gradient: `#667eea` and `#764ba2`
- Text colors and backgrounds
- Look for color values throughout

### Update Content
Edit relevant sections in `index.html`:
- About section
- Academic programs
- News articles
- Staff information
- Events
- FAQ items
